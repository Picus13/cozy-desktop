Class: App

core/app~App()

App is the UI-agnostic entry point of the synchronization client.

UI can do actions and be notified by events via an App instance. For now only the GUI uses it. There used to be a CLI, but it is no longer available because the maintenance costs of having 3 separate packages were too high then.

Constructor

new App()

One should be aware that basePath doesn't mean the same thing everywhere.

The {App} constructor must be given a basePath that points to the parent where the .cozy-desktop/ directory will be created ($HOME by default).

But {App#basePath} actually points to the .cozy-desktop/ directory.
