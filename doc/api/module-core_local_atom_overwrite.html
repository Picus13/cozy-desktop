(inner) ignoreDeletedBeforeOverwritingAdd()

Possibly change event action to 'ignored'.

In case it is a deleted event preceding a replacement. We expect the Merge step to be able to merge the created event as is, even without deleting the document first. However, we want to ignore the deleted event so we don't move the original file to the trash.
